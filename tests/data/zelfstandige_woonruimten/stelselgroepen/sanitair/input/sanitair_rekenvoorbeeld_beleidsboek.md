# Sanitair rekenvoorbeeld beleidsboek

Verwijzing: zelfstandige-woonruimten.md, par. 2.6 (rond regel 971-972).

Het beleidsboek: basis 11 pt (bad/douche 7 + douche slaapkamer 4) + extra 5,75 pt = **16,75 pt** (wastafel niet genoemd).

De code waardeert extra sanitaire voorzieningen alleen als er een wastafel in de badruimte staat. In deze test is daarom een wastafel (1 pt) meegenomen. Totaal in de test: 1 + 7 + 4 + 5,75 = **17,75 pt**.
